# Thai Font Files

Please add the following LINE Seed Sans Thai font files to this directory:

1. `LINESeedSansTH_W_Bd.woff` - Bold weight
2. `LINESeedSansTH_W_Rg.woff` - Regular weight
3. `LINESeedSansTH_W_Th.woff` - Thin/Light weight
4. `LINESeedSansTH_W_He.woff` - Heavy weight
5. `LINESeedSansTH_W_XBd.woff` - Extra Bold weight

## Download Source
You can download LINE Seed Sans Thai fonts from:
- [LINE Seed Font Official Website](https://seed.line.me/index_en.html)

## Usage
These fonts are already configured in `app/globals.css` and will be automatically loaded once the files are placed in this directory.

### Available CSS Classes:
- `.th` - Bold
- `.th-rg` - Regular (default for body)
- `.th-light` - Light
- `.th-heavy` - Heavy
- `.th-xbold` - Extra Bold
- `.th-rg-heading` - Regular with optimized heading spacing
- `.th-rg-body` - Regular with optimized body text spacing
